#########本软件仅提供购买myCobot用户使用############
版本号：V1.2					   #
使用说明：					   #
下载本软件后解压到任意磁盘，点击 myCobot.exe 运行。#
软件运行时会打开终端窗口，请不要关闭。		   #
选择终端，机械臂末端中为Atom，底部为Basic	   #
选择串口，首次使用请保证只有一个外部设备	   #
选择固件，终端固件都是以终端名字开头的命名	   #
点击burn，烧录时终端窗口会反馈进度，显示	   #
Leaving...					   #
Hard resetting via RTS pin...			   #
表示烧录完成。					   #
